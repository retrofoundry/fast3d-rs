# fast3d-rs Examples

This directory contains examples demonstrating how to use fast3d-rs.

## Available Examples

### Triangle
A standalone example showing a rotating triangle with vertex colors.

**Run with:**
```bash
cargo run --example triangle
```

See [triangle/README.md](triangle/README.md) for more details.

## Building Examples

All examples can be built with:
```bash
cargo build --examples
```

Run a specific example:
```bash
cargo run --example <example-name>
```
